Нужно реализовать функцию, которая принимает два аргумента. Необходимо вернуть последнюю цифру
от возведения первого аргумента в степень равную второму аргументу.

При решении отталкивался от очевидного свойства при возведении в степень, что последние цифры 
для каждого показателя степени повторяются с определенной периодичностью (1, 2 или 4).

Examples
```
lastDigit(4n, 1n)            // returns 4n
lastDigit(4n, 2n)            // returns 6n
lastDigit(9n, 7n)            // returns 9n  
lastDigit(10n,10000000000n)  // returns 0n
```
