# Coding Challenges Solutions

Репозиторий содержит решения алгоритмических задач с платформ `Codewars`, `LeetCode` и других ресурсов.

## Структура проекта
Каждая задача размещена в отдельной директории и включает два файла:

```
<platform>/?.<rank>/<task-name>/
├── description.md   # Условие задачи
├── solution.js      # Решение
```

### Пример:
```
codewars/5kyu/Number of proper fractions/
├── description.md
├── solution.js
```

Задачи, решение которых отложено имеют в названии директории `[not not solved]`
## Цели проекта

- Формирование единой базы решений для повторения и анализа
- Синхронизация решений между устройствами
- Помощь знакомым разработчикам, если им лень самим открывать `codewars` :)
- Фиксация удачных решений от сообщества (как правило, размещаются в solution.js в виде `fnNameN()`)

Основные решения представлены на языке `JavaScript`.
Дополнительно могут встречаться задачи, выполненные на `TypeScript` и `React`

## Обновления
Репозиторий пополняется по мере решения новых задач.
